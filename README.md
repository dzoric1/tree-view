# tree-view

## [Ссылка на приложение](https://tree-view-phi.vercel.app/)

## Запуск проекта

### Установка зависимостей

```sh
yarn
```

### Запуск dev режима

```sh
yarn dev
```

или

```sh
yarn serve
```

### Сборка приложения

```sh
yarn build
```

### Запуск prod режима

```sh
yarn preview
```
